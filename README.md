# Procurement Spend Analysis

## Problem Statement
Organizations lack visibility into vendor-wise and category-wise procurement spend.

## Tools Used
- SQL
- Power BI
- Excel

## Key Analysis
- Vendor-wise total spend
- Category-wise spend distribution
- Month-on-month cost trend
- Vendor concentration analysis

## Key Insights
- Top vendors contribute majority of spend
- Certain categories have high cost variability
- Opportunities for vendor consolidation

## Business Impact
This analysis helps management improve cost control and procurement efficiency.
